This is the repo for the Buildertrend Performance Monitoring Header

HOW TO RUN

  1. move into the performacemonitor directory
  2. if you dont already have npm on your system, install it
  3. install the dependency using

    npm install reset-css --save

  4. start the project using

    npm run start

  5. for service worker functionality use

    npm run prod

  6. go to http://localhost:3000/ for nonprod
  7. go to http://localhost:5000/ for prod
